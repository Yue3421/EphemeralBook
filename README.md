# EphemeralBook Project

Project ini adalah aplikasi berbasis Laravel untuk kebutuhan web e-commerce buku secondhand.

## Teknologi
- Laravel 12
- PHP
- MySQL

---

## Cara Menjalankan Project

Ikuti langkah berikut:

```bash
php artisan migrate
php artisan db:seed
```
